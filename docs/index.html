Häufige Fragen

Öffne den Tab Einstellungen und passe den Regler „Tagesziel" auf deine gewünschte Menge an (500–5000 ml). Die Änderung wird sofort übernommen.
Aktiviere unter Einstellungen die Option „Erinnerungen" und wähle Intervall sowie Nicht-stören-Zeiten. Die Benachrichtigungen werden lokal auf deinem Gerät geplant – kein Account nötig.
Alle Daten werden ausschließlich lokal auf deinem Gerät gespeichert. WaterTrack überträgt keinerlei persönliche Daten an externe Server oder Dritte.
Gehe zu Einstellungen und tippe ganz unten auf „Alle Daten zurücksetzen". Dadurch werden alle gespeicherten Daten dauerhaft von deinem Gerät gelöscht.
Dein Streak erhöht sich um 1 für jeden aufeinanderfolgenden Tag, an dem du dein Tagesziel erreichst. Ein verpasster Tag setzt den Streak auf 0 zurück. Bleib dran!
