2 Welche Daten werden erfasst?

WaterTrack speichert ausschließlich folgende Daten lokal auf deinem Gerät:

  • Tägliche Wasserzufuhr (Menge in ml, Zeitstempel)
  • Persönliche Einstellungen (Tagesziel, Erinnerungsintervall, Bechergröße)
  • Streak-Zähler (aufeinanderfolgende Tage mit erreichtem Ziel)

3 Wo werden die Daten gespeichert?

Alle Daten werden ausschließlich lokal auf deinem Gerät über AsyncStorage gespeichert. Es erfolgt keine Übertragung an externe Server, Cloud-Dienste oder Dritte.

4 Welche Daten werden NICHT erfasst?

  • Keine personenbezogenen Daten (Name, E-Mail, Adresse)
  • Keine Standortdaten
  • Keine Weitergabe von Gesundheitsdaten an Dritte
  • Keine Analyse- oder Tracking-Tools (kein Google Analytics, kein Firebase)
  • Keine Werbung oder Ad-Tracking
  • Keine Weitergabe von Daten an Dritte

5 Berechtigungen

Push-Benachrichtigungen (optional): Werden ausschließlich genutzt, um dich ans Trinken zu erinnern. Alle Erinnerungen werden lokal auf dem Gerät geplant – kein externer Dienst ist beteiligt.

6 Datenlöschung

Du kannst alle gespeicherten Daten jederzeit unter Einstellungen → Alle Daten zurücksetzen löschen. Die Deinstallation der App entfernt automatisch alle Daten von deinem Gerät.

7 Kinder

WaterTrack erfasst keine personenbezogenen Daten und ist für alle Altersgruppen geeignet (9+).
